## Branch Naming Convention
- `feature/eda` — Exploratory Data Analysis
- `feature/deployment` — Streamlit deployment
- `feature/preprocessing` — Data preprocessing
- `feature/demo-pages` — Demo and ethics pages

## How to Contribute
1. Create a new branch from `main`
2. Make your changes
3. Commit with a meaningful message
4. Open a Pull Request to `main`
5. Request a review from a teammate

## Commit Message Guidelines
- `Add ...` — for new files
- `Update ...` — for changes to existing files
- `Fix ...` — for bug fixes

## Project Stages
Every member must contribute to:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model building and evaluation
- Deployment
- Documentation
